# Support

CHord is an open source project maintained on a best effort basis.

- Bugs and feature requests: [GitHub issues](https://github.com/orhaugh/CHord/issues), using the
  templates.
- Questions: open an issue describing what you are trying to achieve; include CHord and
  ClickHouse versions.
- Security reports: follow [SECURITY.md](SECURITY.md), never a public issue.
- ClickHouse server questions (SQL, configuration, operations) belong with the
  [ClickHouse documentation](https://clickhouse.com/docs) and community rather than this
  repository.

There is no commercial support. Response times are not guaranteed; well written reports with
reproductions are answered first.
